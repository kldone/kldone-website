BONETOX is a fast-paced arena FPS built for short, intense sessions. Inspired by DOOM and Devil Daggers with brutal, skill-based action.

Players build their arsenal through a weapon skill tree and fight increasingly difficult waves of enemies. The game focuses on fast, skill-based combat and high replayability.

Instead of roguelike randomness, the progression system allows players to deliberately create weapon builds that match their preferred playstyle. Multiple weapon variations and upgrades provide freedom to experiment with different combat strategies.
What Makes It Different

Key Features

  • Melee Resource System. Sledgehammer kills grant armor. Chainsaw kills restore health.
  • Manual Reload with Magazine Waste. Reload too soon and lose your remaining ammo. Tactical decisions matter.
  • Environmental Traps That Evolve. Blood pools become spike traps as waves progress.
  • Gore System with blood decals, bone physics and visceral destruction powered by Unreal Engine 5.
  • Diverse Arsenal from chainsaw to grenade launcher.
  • Temporary Power-Ups that completely change the rules of the fight.
  • Global Leaderboards and skill tree progression.

Development Status

All core gameplay systems are implemented and functional. The game is built primarily in C++ with minimal Blueprint usage for game logic. Core mechanics, combat systems, and performance optimizations are already in place.
